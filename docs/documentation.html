API Documentation

Panduan lengkap untuk menggunakan API kami.

Pendahuluan

Selamat datang di dokumentasi API kami. Di sini Anda akan menemukan semua informasi yang Anda butuhkan untuk berintegrasi dengan layanan kami.

Memulai

Untuk memulai menggunakan API kami, Anda tidak memerlukan kunci API khusus untuk sebagian besar endpoint publik. Cukup buat permintaan HTTP ke endpoint yang relevan seperti yang dijelaskan di bawah ini.

Contoh Permintaan Dasar

Berikut adalah contoh cara membuat permintaan ke salah satu endpoint kami menggunakan JavaScript Fetch API: 


fetch('YOUR_API_ENDPOINT')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Pastikan untuk mengganti 'YOUR_API_ENDPOINT' dengan URL endpoint yang sebenarnya.

Kode Status

API kami menggunakan kode status HTTP konvensional untuk menunjukkan keberhasilan atau kegagalan permintaan API:

  • 200 OK - Permintaan berhasil.
  • 400 Bad Request - Permintaan tidak valid karena parameter yang salah atau hilang.
  • 404 Not Found - Endpoint yang diminta tidak ditemukan.
  • 500 Internal Server Error - Terjadi kesalahan di server kami.
